Importing and Exporting Registries¶

Importing¶

Importing a registry definition saved a yaml file. Login as an admin and navigate to “Importer” under “Admin Page”, and select a previously exported registry yaml file from your file system.

Export¶

Exporting of a defined registry to a yaml file. Login as an admin and navigate to the registry required (“Registries” under “Admin Page”). Select the export custom action and run it.
